About KJV Bible App

This application provides access to the King James Version (KJV) of the Bible in a digital format. Features include:

  • Browse all 66 books of the Bible
  • View chapter-by-chapter content
  • Search across all scripture
  • Clean, readable interface
Technical Details

Built with:

  • Python Flask backend
  • Pandas for data handling
  • Bootstrap 5 for interface
  • KJV text from public domain sources
